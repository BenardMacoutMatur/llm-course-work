📌 Overview

This module covers the fundamentals of Retrieval-Augmented Generation (RAG) and agentic RAG pipelines. We index course documents, build a RAG assistant, apply chunking to reduce token usage, and extend the pipeline into an agentic loop where the LLM decides when and what to search.

📚 Topics Covered

  • Indexing documents with minsearch using text and keyword fields
  • Building a RAG pipeline using RAGBase and extending it with AgenticRAG
  • Chunking long documents with a sliding window for more precise retrieval
  • Measuring LLM token usage via the OpenAI Responses API
  • Building an agentic loop with toyaikit where the LLM controls search
